# CheckMachine
一阶段实战项目-仓储盘点机：项目代码**只提供思路**，**仅供参考**
建议同学们脚踏实地自己实现各功能（要不然二阶段会很痛苦）

## 学习技术点
C/C++基础、指针、链表、文件读写、文件流操作、面向对象OOP
**STL数据结构:**
-Map容器
-Vector容器

## 项目实现主要功能
新增商品、删除商品、商品出入库、商品检索、盘点冲正

12个工作日
